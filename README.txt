Project Description
-------------------
This project aims to automate the generic problem of sorting and delivering objects. 
Sorting is done by a gripper bot, using RFiD tags to identify an object, communicating with a coordinator using zigbee to find a delivery bot.
The delivery bot then delivers the object to the appropriate destination.
All the bots are moving in the same arena, and hence path planning and collision avoidance is also an integral part of the project.
